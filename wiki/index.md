## Read these before trusting a name
- [The Tate / TateAI / Terminal namespace](concepts/tate-tateai-terminal-namespace.md) — **critical.** Five different things share overlapping names: the real Andrew Tate, `$TateAI` (this project's token), `@tate_terminalbot`/"tate terminal" (a forwarded-content source, not a person), a chat sender literally named "Tate AI" (an unrelated rival `$RNT` shill-bot), and this bot itself (Terminal Subagent).
- ["Deleted Account" is not a person](concepts/deleted-accounts.md) — the single highest-volume "sender" (33,803 messages) is Telegram's placeholder for anyone who'd deleted their account by export time, not one individual.
